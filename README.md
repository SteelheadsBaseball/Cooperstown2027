# Steelheads 11U — Cooperstown 2027

Family-facing planning hub for the Steelheads 11U travel baseball team's potential 2027 Cooperstown trip.

**Live site:** https://steelheadsbaseball.github.io/Cooperstown2027/

## What this is

A simple static site with:

- Dreams Park vs. CASV comparison
- Per-family budget estimates
- Fundraising ideas
- Rough timeline
- FAQ for families

Built to be read on phones at the ballpark. Unlisted (`noindex` meta tag) — share the link with team families only.
